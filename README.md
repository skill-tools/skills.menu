# skills.menu

Website for the [Agent Skills](https://agentskills.io) ecosystem and [skill-tools](https://github.com/skill-tools/skill-tools) toolkit.

**Live:** [skills.menu](https://skills.menu)

## Stack

- [Astro](https://astro.build) 5.x -- static site generator
- WebGL background -- triangulated mesh network
- Zero JavaScript frameworks -- vanilla Astro components

## Development

```bash
npm install         # Install dependencies
npm run dev         # Start dev server at localhost:4321
npm run build       # Build to ./dist/
npm run preview     # Preview production build
```
