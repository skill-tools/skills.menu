What's a SKILL.md?

A portable format for AI agent instructions. YAML frontmatter + Markdown body.

deploy-vercel/SKILL.md
---
name: deploy-vercel
description: Deploy web apps to Vercel.
version: 1.0.0
---

# Deploy to Vercel

## Prerequisites
- Vercel CLI installed (`npm i -g vercel`)

## Steps
1. Run `vercel --prod`
2. Verify deployment URL
3. Report live URL to user
frontmatter

YAML metadata for discovery. Name must match directory.

description

Tells agents when to use this skill. Used for routing.

body

Markdown instructions. Keep under 5,000 tokens.

resources

Reference scripts/, references/, assets/ for progressive disclosure.
