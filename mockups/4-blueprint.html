// SKILL.md format

yaml frontmatter + markdown body. portable agent instructions.

deploy-vercel/SKILL.md
---
name: deploy-vercel
description: Deploy web apps to Vercel.
version: 1.0.0
---

# Deploy to Vercel

## Prerequisites
- Vercel CLI (`npm i -g vercel`)

## Steps
1. Run `vercel --prod`
2. Verify deployment URL
3. Report URL to user
// frontmatter

yaml metadata. name must match directory.
